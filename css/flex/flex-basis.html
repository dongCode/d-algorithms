Flex Basis Example

The flex-basis property is used to set the initial main size of a flex item.

It can be set to a fixed length (e.g. 100px), a percentage (e.g. 50%), or auto.

If the flex-basis value is set to auto, the size of the item will be based on its content.

Usage Notes

  • The flex-basis property is used in conjunction with flex-grow and flex-shrink to determine how flex items are sized and how they respond to available space.
  • When using flex-basis with a fixed length or percentage value, be aware that the item may overflow its container if there is not enough space available.
  • When using flex-basis: auto, the size of the item will be based on its content. However, if the item is too wide or too tall to fit in the available space, it will overflow its container.
  • When using flex-basis: auto, the min-width and max-width properties can be used to set minimum and maximum sizes for the item.
